### 1. Fluid-solid jump conditions (paper Eq 10-11)

The order-0 jump matrices for the 2D fluid-elastic interface
along a parametric curve Γ(τ) with tangent t = (x', y') and
normal n = (-y', x'):

- **C₁⁰**: 2×3 matrix on fluid side (couples normal velocity
  v·n and pressure p)
- **C₂⁰**: 2×5 matrix on solid side (couples normal velocity
  v·n and normal stress σ·n)
- **L₁⁰**: 1×3 matrix (boundary conditions on fluid side —
  per paper Eq 11, empty for fluid-solid case)
- **L₂⁰**: 1×5 matrix (vanishing tangential shear stress on
  solid side)

Each matrix entry transcribed under **side-by-side user-
confirmed review** per the project-wide transcription protocol
(see `feedback_transcription_workflow_arxiv_then_side_by_side`
memory entry).

### 2. Compatibility conditions (paper Eq 21, 24)

Recursive differentiation produces compatibility constraints
between spatial derivatives of U:
- Eq 21: fluid vorticity = 0 (consequence of perfect fluid)
- Eq 24: solid stress-tensor symmetry

These reduce the number of independent components in U^k from
3(k+1)(k+2)/2 (fluid) or 5(k+1)(k+2)/2 (solid) to a smaller
set via the V^k = G^k V^k transformation.

### 3. SVD-derived nullspace K (paper Eq 26)

K^k_i is the kernel of L^k_i G^k_i, computed via SVD. Encodes
which independent components of U^k can be chosen freely
without violating the boundary condition.

### 4. Modified-value projector (paper Eq 42)

The ESIM modified value at irregular point M(I, J) ∈ Ω₂:
```
U*_{I,J} = Π^k_{I,J} · G^k_2 · K^k_2 · M̄⁻¹ · (U^n)_B
```
where:
- Π^k_{I,J}: Taylor-basis at (I, J) (Eq 32)
- M̄⁻¹: least-squares restriction inverse of the Taylor-fit
  matrix M (Eq 38-40)
- (U^n)_B: numerical values at neighborhood disk B around the
  orthogonal projection P of M on Γ

### 5. Time-stepping near interfaces (paper Eq 43-44)

At irregular point (i, j) belonging to medium Ω(i, j):
- Stencil reads from cells on the SAME side: use numerical
  values U^n
- Stencil reads from cells on the OTHER side: use modified
  values U*
- Apply the standard scheme H to the mixed Ũ array
